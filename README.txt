To record and plot serial data in real time using python:

Install updated python3.x
	This includes pip which significantly helps the installation process. Lets you install PyPl

___________________________________________________________________________________________________
LibraryNeeded: description
	install:(command prompt to install)
---------------------------------------------------------------------------------------------------

pyserial:Allows python to read from the serial port. Only one program can be reading the port
	install: pip install pyserial

matplotlib: Allows python to make graphs. Lots of documentation out there for it.
	install: pip install matplotlib

drawnow: Allows for the continuously updating graph
	install: pip install drawnow
___________________________________________________________________________________________________

Once these are installed, the program should run and record data to a text file.
